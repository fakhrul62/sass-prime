# Prime

Prime is a responsive workplace signal intelligence website built with
Next.js, React, and TypeScript.

## Getting Started

Install dependencies and start the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Commands

```bash
npm run dev
npm run lint
npm run build
npm run start
```

## Routes

- `/`
- `/features`
- `/about`
- `/how-it-works`
- `/resources`
- `/demo`
- `/checkout`

The contact, newsletter, and checkout endpoints store local development records
under `data/`. These files are intentionally excluded from Git.

## Design Direction

`REFERENCE_ENGINEERING_STUDY.txt` contains the reusable typography, color,
layout, motion, accessibility, and performance guidelines followed by the
project.

## Deployment

The production project is connected to the GitHub `main` branch and deployed
through Vercel.
